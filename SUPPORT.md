# Support

This app and the Stream Panel profiles were created in free time.

The app remains usable without payment. If a support message appears after repeated use, it is a voluntary reminder only and does not block the application.

If you find the project useful, you can support the author through the support link or payment method provided in the repository or app.

## Reporting Issues

When reporting a problem, include:

- app version
- Windows version
- Stream Panel GK150 version if known
- what action was running
- the visible error message or log output
- whether Stream Panel was open during import/repair

For profile navigation problems, mention which scene/page/button is wrong.
